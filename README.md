# Portfolio Planner

A scenario-based investment planning app built with React + Express + MySQL.

## Live App

**URL:** https://portfolio-app-production-04a6.up.railway.app

## Tech Stack

- **Frontend:** React + Vite + TypeScript + TailwindCSS
- **Backend:** Express.js + tRPC
- **Database:** MySQL (Railway-hosted)
- **Hosting:** Railway (Hobby plan)

## Features

- Scenario-based portfolio planning with multiple portfolios
- S&P 500 stock library with sector filtering
- Drag-and-drop stock allocation with smooth sliders
- Cloud sync — data persists across sessions
- Analytics view with projection charts
- Single-user mode (no login required)

## Deployment

Auto-deploys on every push to the `main` branch via Railway GitHub integration.
